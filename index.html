About Orbix (ORX)

Orbix is a next-gen upgradeable ERC20 token with staking, transaction fees, and rewarding features built on Polygon. Designed for scalability, security, and community-driven growth, ORX empowers holders with passive income through staking pools and transaction rewards.

  • 🪙 Total Supply: 900 Million ORX
  • ⚡ Transaction Fee: 2% sent to the burn address
  • 🔒 Staking Pools with fixed returns
  • 🔄 Fully Upgradeable contract architecture
  • 🌐 Compatible with major wallets: MetaMask, Trust Wallet, WalletConnect
